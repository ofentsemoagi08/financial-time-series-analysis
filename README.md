# Financial Time Series Analysis

## Objective
Analyse Amazon stock price data to compute returns, volatility, and trend behaviour, deriving insights relevant for financial decision-making.

## Dataset
- Source: Yahoo Finance
- Asset: AMZN
- Period: Last 5 years
- Frequency: Daily

## Tools
- Python (pandas, numpy, matplotlib)
- yfinance

## Key Findings
- Volatility clusters during periods of market stress
- Short-term vs long-term volatility shows risk differences
- Drawdowns highlight downside risk exposure
